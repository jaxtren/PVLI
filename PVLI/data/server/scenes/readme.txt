Blender export settings:
  glTF (separate .gltf + .bin + textures)
  include: custom properties, punctual lights
  disable +Y Up transform
  enable all geometry settings

Light - Custom Properties (supported by Lighthouse):
  radius (point / spot light)
  angle (directional light)
  (example scene: pica)
  
Textures:
  Lighhouse2 doesn't support bump maps (only normal maps)
